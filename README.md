# MCP Paper Search

## 项目简介

一个基于MCP协议的论文搜索工具，支持通过关键词、作者、期刊、会议等条件进行搜索，并返回搜索结果的摘要、关键词、引用次数、下载链接等信息。

## 项目结构

```
mcp-paper-search
├── build/
├── src/
├── package.json
├── tsconfig.json
├── README.md
```
